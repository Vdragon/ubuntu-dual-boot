![「更新管理員」視窗，提示訊息說有新 Ubuntu 發行版 '14.04' 可供升級](./額外螢幕擷圖/8-upgrade.png)

如果你要從 Ubuntu 12.04 升級到 14.04 ，可以從「更新管理員」裡面按「升級」來完成。這個升級按鈕會在 7 月 24 日， Ubuntu 14.04.1 發行時出現。

如果要透過命令列升級，請用 "sudo do-release-upgrade" 。
